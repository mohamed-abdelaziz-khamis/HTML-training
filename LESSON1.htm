In HTML the basic element is the tag

To open a tag use:

<TagName>

to close a tab use:

</TagName>

For example if you want to write Boldtext

<b>This Text is Bold</b >

when you want to assign a value to a property in a tag use

<TagName Property1 = “value1” property2 = “value2”></ TagName >

For example if you want to write Large text in Red use

<font color =”red” size = “5”> Large text in Red< /FONT >

·        HTML documents have two parts, the head and the body.

o       The head element provides information that does not affect the rendering of the document but could be of use to the browser.

o       The body is the larger part of the document, as the body of a letter you would write to a friend would be.

·        The to parts are represented in code like that


<html> <!----------page start -------------->
            <head>
                        <!---------- head part -------------->
            </head>

            <body>
                        <!----------body part -------------->
            </body>
</html><!----------page end -------------->

The following tags are valid in HEAD element:

Title

Contains the title of the document.
example

<title>page title</title >


Base

Specifies an explicit URL used to resolve links and references to external sources such as images and style sheets.
Example:

<base HREF = ” http://www.myweb.com/ “ />


BGSound

Enables an author to create pages with background sounds or soundtracks.

Example < /U >

<bgsound src =”sounds/amrdiab.midi” loop = 1>

BaseFont

Sets a base font value to be used as the default font when rendering text.

Example

<BaseFont size = 4>

Meta

The <META> tag can be used for a few different purposes. Usually, you should place the <META> tag within the <head> tags at the beginning of your document.

To improve search engine results, we will use two specific attributes within the meta tag.

Example:

<META name="description" content="description of page goes here">
<META name="keywords" content="keywords go here">


Refresh a page

Example:

<META HTTP-EQUIV="Refresh" CONTENT="10">
<!--this will auto refresh the page every 10seconds-->
