Frames 
Frames 
is a feature which allows a single browser window to be divided into multiple 
sections, each with an independent HTML page loaded inside it, and these HTML 
pages can interact with each other. Each page loaded within each section of the 
frames window is a separate HTML document.
Remarks


·        The FRAMESET element is 
a container for the FRAME element.


·        An HTML document can 
contain either the FRAMESET element or the BODY element. As shown in the 
following example


·        HTML document included in 
<noframes> are for browsers not supported frames
 




<HTML>
<HEAD>
<title> frames….</title >
</HEAD>
<FRAMESET COLS ="25%, 50%, *">
<FRAME SRC ="contents.htm">
<FRAME SRC ="info.htm">
<FRAME SCROLLING="NO" SRC  ="graphic.htm">
</FRAMESET>
<NOFRAMES> Your browser doesn’t support frames </NOFRAMES >
</HTML>


 




Basic Attributes <frameset>
Border
the space between the frames, including the 3-D border.
BorderColor
The color of the border 
Cols
the frame widths of the object
values
 "50, 2*, *, 80"
 "40%, 60%"
FrameBorder
whether to display a border for the frame.
FrameSpacing
the amount of additional space between the frames.
Rows
the frame height of the object
values
 "50, 2*, *, 80"
 "40%, 60%"
 




Basic Attributes <frame>
BorderColor
the border color of the object.
FrameBorder
whether to display a border for the frame
Height
the height of the object
Name
The frame name
Remark
this property is important because this name used as window name in the target property
MarginHeight
the top and bottom margin heights before displaying the text in a frame.
MarginWidth
the left and right margin widths before displaying the text in a frame.
NoResize
whether the user can resize the frame
Scrolling
whether the frame can be scrolled
Values
yes, no, auto(default)
Src
A URL to be loaded by the object.
