Marquee <Marquee>

Usage: Creates a scrolling text marquee 

Basic Attributes

Align

The alignment of the object

Values
Right, left, center

Behavior

how the text scrolls in the marquee

Values

Scroll : The text scrolls off the end and starts over.
Alternate: Marquee scroll direction reverses when its content reaches the edge of the container.
Slide: The text scrolls to the end and stops.

BgColor

Specifies the back ground color

Direction

Specifies the direction of moving

Width

Specifies the width

Height

Specifies the height

Hspace

Specifies the horizontal spacing

Vspace

Specifies the vertical spacing 

Loop

Specifies the number of times the marquee will play

ScrollAmount

the number of pixels the text scrolls between each subsequent drawing of the marquee.

ScrollDelay

The speed of the marquee
