HTML stands for Hyper Text Markup Language, the major language of the Internet's World Wide Web. Web Sites and Web Pages are written in HTML. With HTML you have the ability to bring together text, pictures, sounds, and links... all in one place!

·        I prefer to make a special folder for every site for example “\myweb”

o       Try to put images, sounds, CSS… in separated folders for example put images in a sub folder “\images” ,sounds in sub folder “\sounds”  …

·        HTML is not made specifically for any platform (i.e. Windows PC or MAC)

·        HTML can be displayed on any computer with a web browser

·        All of your HTML files must be saved in standard (ASCII) text format

o       You can use notepad to write your code and save the file as .htm or .html

o       You can use any html editor to write html code like Visual Studio .NET, Visual InterDev, Dream waver, FrontPage …. If you used any of these editors it’s easiest for you to find errors, the code will automatically be colored, the code will automatically be aligned and it automatically be completed.

·        A very important note is to make the home page of your site titled “index.htm”.
because this is the way almost all web servers are set up to handle. (You can check with the host/web server if you can't get index.html to work. Some systems use other filenames, for example: default.html, index.htm , default.htm)

In most cases you will be provided web page space for free. A free host will give you a decent amount of file space to place your web pages and graphics.
 HTML files are very small, and the graphics formats supported on the world wide web (.gif and .jpg) are compressed formats, which means they are saved smaller and more efficiently.

Say you wrote a link page, called links.html on your hard drive, saved as c:\mysite\links.html. Then you decide to link it from index.html as c:\mysite\links.html. You browse your files on your hard drive, and the link works fine. Then you upload it to the web, and get one of your friends to try. When they click the link, they will get an error saying "File Not Found"! Why? Because there is no c:\mysite\links.html on their hard drive!… how can I solve it? Make all your links relative how? put all the site in the same folder then just use “links.html” so the host will find the file in your folder there.
