Paragraph  <p>

Break Line <br>

Usage: When we want to Insert a line break.

Usage: to Render text without line breaks .

Centering <center>

Usage: When we want to center subsequent text and images

Division <div>

Usage: When we want to Specify a container that renders HTML we use the tag

Listing Items

UL : used to Denote an unordered list <ul><li>...</ul>

OL : used to Denote an ordered list <ol><li>...</ol>

DL: used to Denote a definition list <dl compact><dt>...<dd>...</dl>

Customized Listing Items

We can customize the list by setting the TYPE attribute in the OL tag to one of the following values (string).

 

1

Default. Associate numbers with each item in an ordered list.

a

Associate lowercase letters with each item in an ordered list.

A

Associate uppercase letters with each item in an ordered list.

i

Associate small Roman numerals with each item in an ordered list.

I

Associate large Roman numerals with each item in an ordered list.

disc

Associate a solid disc with each item in an unordered list.

circle

Associate a hollow circle with each item in an unordered list.

square

Associate a solid square with each item in an unordered list.
